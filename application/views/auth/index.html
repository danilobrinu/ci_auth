CI_Auth, is a set of libraries, that contains libraries for handling authentication, inspired by Django authentication. These libraries are Auth, Login_required, Permission_required, Group_required, Pass_test_required and Syncdb.

Auth Library

This library contains three methods, these methods are authentication, login and logout.

  • The "authentication" method authenticates the user and returns an array of user data to be used in the session.
  • The "login" method creates a session with the data returned by the authentication method. Default redirects to main, but can configure.
  • The "logout" method destroy session created by the login method. Default redirects to **admin_site/login **, but can configure.

Login_required Library

This library is used to restrict access to unauthenticated users, and the redirects to main. But, can configure. If used to restrict views, in where need authentication.

Permission_required Library

This library is used to restrict access to unauthorized users if have not permission or permissions. Contain two methods has_permission and has_permissions.

  • The "has_permission", restricts access if you have not permission.
  • The "has_permissions", restricts access if you have not permissions, one and more permissions.
  • The two methods redirects to "main" if restriction is false, but can configure.

Group_required Library

This library is used to restrict access to authorized users if have not group or groups. Contain two methods has_group and has_groups.

  • The "has_group", restricts access if you have not group.
  • The "has_groups", restricts access if you have not groups, one and more groups.
  • The two methods redirects to main if restriction is false, but can configure.

Pass_test_required Library

This library is used to restrict access to authorized users if not pass test with method test, redirects to main if restriction is false, but can configure.

Syncdb Library

This library contains methods to create the tables of authentication, used by the librerias of CI_Auth.
